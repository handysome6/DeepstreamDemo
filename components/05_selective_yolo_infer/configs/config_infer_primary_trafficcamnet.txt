# P0.5 default detector config.
#
# Uses the ResNet18 TrafficCamNet primary detector that ships with DeepStream
# 9.0 at /opt/nvidia/deepstream/deepstream-9.0/samples/models/Primary_Detector.
#
# Why this is the *default*: the architecture rule is that a component must
# stand up on a fresh checkout without external downloads. TrafficCamNet is
# already inside the container image and produces real bounding boxes on
# people / vehicles, which gives nvdsosd something visible to draw and is
# enough to prove the per-stream "selective inference" shape.
#
# To swap to YOLO once the model is staged, copy this file and change the
# model paths plus the parser block. See configs/config_infer_primary_yolov8.txt
# for the YOLO slot.
